# CleanWipe

CleanWipe is a tool that listens clipboard data and cleans up unwanted data from it, such as url utm labels, fbclid, etc.

### Quickstart

```bash
$ git clone https://github.com/damirm/clipwipe
$ cd clipwipe
$ make
$ out/clipwipe
```
